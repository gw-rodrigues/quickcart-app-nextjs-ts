## ðŸ›  Instalation

</div>

Let's start cloning the app:

```bash
git clone https://github.com/gw-rodrigues/reponame.git
```

Move to app folder:

```bash
cd repo
```

Install dependencies:

```bash
npm install
```

Start the app:

```bash
npm run dev
```

Go to [http://localhost:3000](http://localhost:3000) and enjoy.
